# 全景图像拼接系统简介 python1107

## 项目概述

全景图像拼接系统是基于 Python 语言，采用 OpenCV 与 Django 框架实现的。该系统支持用户登录、个人信息展示、用户信息管理、密码修改以及图片拼接和图像管理等核心功能。

## 技术栈

- **开发语言**: Python
- **数据库**: MySQL
- **框架**: Django
- **开发工具**: PyCharm, Navicat

## 功能模块

### 系统角色

- **系统登录**: 用户身份验证
- **首页**: 全景图像拼接效果展示
- **个人信息展示**: 用户资料展示与编辑
- **用户信息管理**: 管理员对用户信息进行操作
- **修改密码**: 用户可以修改个人密码
- **图片拼接功能**: 支持用户上传图片并进行拼接处理
- **图像管理功能**: 管理拼接后的全景图像

## 运行环境

- Python 环境推荐版本：3.6 或以上
- MySQL 数据库版本：5.7 或以上
- 操作系统：兼容 Windows、Linux 和 macOS

## 部署步骤

1. 安装 Python 环境及相关依赖库
2. 安装并配置 MySQL 数据库
3. 导入项目所需的数据表和初始数据
4. 运行 Django 项目，并根据配置文件进行设置
5. 启动服务器，完成部署
